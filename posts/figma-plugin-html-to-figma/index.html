Html 轉 Figma 的小工具 Figma Plugin匯入現有網站

Posted Aug 15, 2021
By chenuin
1 min read

Figma 如何匯入現有網站，可以使用 Figma to HTML, CSS, React & more! 快速解決！

可以先安裝 google chrome 的擴充功能、Figma的擴充功能，再到網站使用 google chrome 的擴充功能匯出 Json，並到 Figma 將檔案匯入。

成果雖然會和實際網站有些差別，但很方便。

  • Figma的擴充功能
  • Google Chrome 的擴充功能
